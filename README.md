# eps-plugin
A plugin in HTML to display English word phonetic symbols 


## How to Install
In `.html` code editor, `<head>` label add   
```
<script src="https://cdn.jsdelivr.net/gh/Wk-P/en-phonetics@main/en-phonetics.js"></script>
```

## How to use
In `.html` code editor, `<body>` label add
```
<en-word>apple<en-word>
```
